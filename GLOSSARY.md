# Глоссарий проекта AppMonitor

| Термин | Расшифровка | Описание |
|--------|-------------|----------|
| **M** | Monitor | Ядро мониторинга — механизм опроса активных окон (win32 API), сбор статистики, работа с БД. Находится в `core/`. |
| **MUI** | Monitor User Interface | Десктопное приложение AppMonitor на PyQt5 — интерфейс пользователя для просмотра статистики, настройки лимитов, уведомлений. Находится в `ui/`. |
| **MWS** | Monitor Web Server | Веб-сервер (FastAPI) для удалённого доступа к данным мониторинга. Находится в `api/`. |
| **AUI** | Admin User Interface | Интерфейс администратора для удалённого управления — просмотр статистики пользователей, управление лимитами, блокировки. Реализован как отдельное клиентское приложение в `client/`. |
| **UPD** | Updater | Модуль автоматического обновления приложения — проверка новых версий, загрузка и установка обновлений. Находится в `updater/`. |
